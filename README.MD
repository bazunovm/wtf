# WTF

WTF — CLI инструмент, который объясняет ошибки команд Linux/Mac, используя правила (YAML) и сохраняет память прошлых решений.

## Установка

1. Клонировать репозиторий:
```bash
git clone <repo-url>
cd wtf
2. Установить зависимости
go mod tidy
3. Собрать бинарник
go build -o wtf ./cmd/wtf
4. Использование
./wtf make
